Introdução ao MVC

O padrão MVC(Model View Controller) é uma arquitetura software utilizada no desenvlvimenro de aplicações para separar a lógica de negócios, a interfaces de usuários e o controle dos dados.

Componentes do MVC

  • Model(Modelo):Representa os dados e a lógica de negócios da aplicação.
  • View(Visão): Responsável pela inteface com o usuário e exibição de dados.
  • Controller(Controlador):Intermedia a comunicção de entre Model e View.
