Model(Modelo)

O Model representa a camada de dados da aplicação. Ele gerencia a lógica de negócios as regras, e a interação com o banco de dados.

Principais funções do Model

  • Armazena e gerencia dados
  • Executa regras de negócios
  • Interage com banco de dados

Exemplo

Imagine um sistema de usuários onde o Model contém as informações com o nome,e-mail e senha, além das regras de validação.
